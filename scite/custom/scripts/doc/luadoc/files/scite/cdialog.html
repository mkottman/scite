File ./scite/cdialog.lua

(Linux only) Completion dialog utilities for the scite module. This module requires utils/completion.rb. There are several option variables used: PLATFORM: OS platform (linux or windows). FILE_OUT: Location of the temporary file that will contain output for various operations. REDIRECT: The command line symbol used for redirecting STDOUT to a file. RUBY_CMD: The command that executes the Ruby interpreter.

choose (option_list, title, show_list_opt)
Displays list or dropdown dialog of choices from a string of options (delimited by newlines). This is generally called by the choose_key or choose_value functions because passing a table of options can be more intuitive.

Parameters

  • option_list: String of options delimited by newlines.
  • title: The title of the completion dialog.
  • show_list_opt: Boolean flag indicating whether the entire list of options is shown as opposed to a dropdown selection.

Return value:

Selection, nil if cancelled, or false if empty string was selected. See also: choose_key , choose_value
choose_key (options, title, show_list)
Shows a completion dialog with choices provided by the keys in a given table of options.

Parameters

  • options: Table of options with keys being the choices.
  • title: The title of the completion dialog.
  • show_list: Boolean flag indicating whether the entire list of options is shown as opposed to a dropdown selection.

Return value:

Choice selected or nil.
